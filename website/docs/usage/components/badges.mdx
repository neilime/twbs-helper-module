<TabItem value="source" label="Source">

```php
<?php

// H1
echo '<h1>Example heading '.$this->badge('New').'</h1>';
// H2
echo '<h2>Example heading '.$this->badge('New').'</h2>';
// H3
echo '<h3>Example heading '.$this->badge('New').'</h3>';
// H4
echo '<h4>Example heading '.$this->badge('New').'</h4>';
// H5
echo '<h5>Example heading '.$this->badge('New').'</h5>';
// H6
echo '<h6>Example heading '.$this->badge('New').'</h6>';

echo PHP_EOL.'<br/>';

// Button
echo $this->formButton(
    [
        'options' => [
            'label'   => 'Profile '.$this->badge('9', 'light').'<span class="sr-only">unread messages</span>',
            'variant' => 'primary',
        ],
    ]
);

```

## Contextual variations

[Twitter bootstrap Documentation](https://getbootstrap.com/docs/4.5/components/badge/#contextual-variations)

<Tabs>
<TabItem value="result" label="Result" default>
  <HtmlCode bootstrapVersion="4.5">
      <div dangerouslySetInnerHTML={{ __html: `<span class="badge badge-primary">Primary</span>
<span class="badge badge-secondary">Secondary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-danger">Danger</span>
<span class="badge badge-warning">Warning</span>
<span class="badge badge-info">Info</span>
<span class="badge badge-light">Light</span>
<span class="badge badge-dark">Dark</span>` }}></div>
  </HtmlCode>
</TabItem>
<TabItem value="source" label="Source">

```php
<?php

foreach ([
    'primary',
    'secondary',
    'success',
    'danger',
    'warning',
    'info',
    'light',
    'dark',
] as $sVariant) {
    echo $this->badge(ucfirst($sVariant), $sVariant);
}

```

</TabItem>
</Tabs>
