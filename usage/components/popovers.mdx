## Example

[Twitter bootstrap Documentation](https://getbootstrap.com/docs/5.3/components/popovers/#example)

<Tabs>
<TabItem value="result" label="Result" default>
  <HtmlCode bootstrapVersion="5.3">
      <div dangerouslySetInnerHTML={{ __html: `<button class="btn btn-danger btn-lg" data-bs-content="And here's some amazing content. It's very engaging. Right?" data-bs-original-title="Popover title" data-bs-toggle="popover" name="button" title="" type="button" value="">Click to toggle popover</button>` }}></div>
  </HtmlCode>
</TabItem>
<TabItem value="source" label="Source">

```php
<?php

echo $this->formButton()->renderSpec([
    'options' => [
        'label' => 'Click to toggle popover',
        'variant' => 'danger',
        'popover' => "And here's some amazing content. It's very engaging. Right?",
        'size' => 'lg',
    ],
    'attributes' => ['title' => 'Popover title'],
]);
```

### Dismiss on next click

[Twitter bootstrap Documentation](https://getbootstrap.com/docs/5.3/components/popovers/#dismiss-on-next-click)

<Tabs>
<TabItem value="result" label="Result" default>
  <HtmlCode bootstrapVersion="5.3">
      <div dangerouslySetInnerHTML={{ __html: `<a class="btn btn-danger btn-lg" data-bs-content="And here's some amazing content. It's very engaging. Right?" data-bs-original-title="Dismissible popover" data-bs-toggle="popover" data-bs-trigger="focus" role="button" tabindex="0" title="">Dismissible popover</a>` }}></div>
  </HtmlCode>
</TabItem>
<TabItem value="source" label="Source">

```php
<?php

echo $this->formButton()->renderSpec([
    'name' => 'popover',
    'options' => [
        'tag' => 'a',
        'label' => 'Dismissible popover',
        'variant' => 'danger',
        'popover' => [
            'trigger' => 'focus',
            'content' => "And here's some amazing content. It's very engaging. Right?",
        ],
        'size' => 'lg',
    ],
    'attributes' => ['title' => 'Dismissible popover', 'tabindex' => '0'],
]);
```

</TabItem>
</Tabs>
